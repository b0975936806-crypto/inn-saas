# InnSaaS 民宿管理系統 - 預訂流程設計稿

> 版本：v1.0  
> 更新日期：2026-03-08  
> 頁面：預訂流程（Booking Flow）

---

## 1. 流程概述

### 1.1 流程目標
- 提供順暢、直覺的預訂體驗
- 減少使用者決策負擔，分步驟引導完成預訂
- 建立信任感，清楚呈現價格與政策

### 1.2 使用者流程
```
選擇日期房間 → 填寫資料 → 確認付款 → 預訂結果
    (Step 1)      (Step 2)     (Step 3)    (結果頁)
```

### 1.3 流程特點
- **進度指示器**：清楚標示目前步驟與剩餘步驟
- **資料保存**：每步驟自動保存，重新整理不遺失
- **返回機制**：可隨時返回上一步修改
- **價格透明**：即時顯示總價格，無隱藏費用
