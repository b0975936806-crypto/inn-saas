# InnSaaS 民宿管理系統 - 房型列表頁設計稿

> 版本：v1.0  
> 更新日期：2026-03-08  > 頁面：房型列表（Room List）

---

## 1. 頁面概述

### 1.1 頁面目標
- 讓使用者瀏覽所有可預訂房型
- 提供篩選功能快速找到符合需求的房型
- 清晰呈現房型資訊協助決策

### 1.2 使用者流程
```
進入列表頁 → 查看/調整篩選條件 → 瀏覽房型卡片 → 點擊有興趣的房型 → 進入詳情/預訂
```

### 1.3 頁面結構
```
┌─────────────────────────────────────────────────────────────┐
│  Navigation                                                 │
├─────────────────────────────────────────────────────────────┤
│  [搜尋篩選區] 日期 · 人數 · 價格區間 · 設備篩選              │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────┬─────────────────────────────┐ │
│  │  篩選側欄 (Desktop)     │  結果區域                    │ │
│  │  - 價格區間             │  - 結果統計                  │ │
│  │  - 設備設施             │  - 排序選項                  │ │
│  │  - 房型類型             │  - 房型卡片網格               │ │
│  │  - 評分篩選             │  - 分頁/載入更多             │ │
│  └─────────────────────────┴─────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│  Footer                                                     │
└─────────────────────────────────────────────────────────────┘
```

## 4. 分頁設計

### 4.1 分頁元件

#### 結構
```
        ┌─────────────────────────────────────────┐
        │  ← 上一頁   1  2  3  ...  10   下一頁 →  │
        └─────────────────────────────────────────┘
```

#### 樣式
- 位置：置中
- 上方間距：48px
- 下方間距：64px

**頁碼按鈕**
| 狀態 | 樣式 |
|------|------|
| 預設 | 40px × 40px、圓形、white 背景、`gray-700` 文字 |
| Hover | `gray-100` 背景 |
| 當前 | `primary-600` 背景、white 文字 |
| 禁用 | `gray-300` 文字、cursor not-allowed |

**導航按鈕**
- 文字：「上一頁」「下一頁」
- 包含箭頭圖示
- 樣式同頁碼按鈕

### 4.2 載入更多（替代方案）

```
┌─────────────────────────────────────────┐

│         [    載入更多房型    ]          │
│                                         │
```

- 按鈕：Outline、Size Large
- 進度文字：14px、`gray-500`、上方間距 16px

## 8. 互動規範

### 8.1 篩選互動
- 選擇篩選條件後即時更新結果（Debounced 300ms）
- URL 同步更新，支援分享篩選結果
- 顯示載入動畫避免閃爍

### 8.2 卡片 Hover
```css
.room-card {
  transition: transform 200ms ease, box-shadow 200ms ease;
}
.room-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}
```

### 8.3 圖片輪播
- 自動輪播：停用
- 滑動切換：支援觸控滑動
- Hover 顯示切換箭頭
