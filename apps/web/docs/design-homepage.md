> 版本：v1.0  
> 更新日期：2026-03-08

## 1. 頁面概述

### 1.1 頁面目標
- 建立品牌第一印象，傳達民宿溫馨專業形象
- 提供快速搜尋入口，讓使用者立即開始預訂流程
- 展示精選房型與特色，激發預訂慾望

### 1.2 使用者流程
```
進入首頁 → Hero 區塊瀏覽 → 使用搜尋 → 瀏覽精選房型 → 點擊房型詳情 → 開始預訂
```

### 1.3 響應式策略
| 裝置 | 佈局重點 |
|------|----------|
| **Mobile** | 單欄堆疊、搜尋框垂直排列、卡片全寬 |
| **Tablet** | 雙欄房型卡片、搜尋框水平排列 |
| **Desktop** | 三欄房型卡片、大圖展示、水平導航 |

## 4. 互動規範

### 4.1 導航滾動效果
```
初始狀態：
- 背景：transparent
- 文字：white

滾動後（scroll > 50px）：
- 背景：white
- 陰影：shadow-sm
- 文字：gray-700
- 過渡：300ms ease
```

### 4.2 卡片 Hover
```css
.room-card {
  transition: transform 300ms ease, box-shadow 300ms ease;
}
.room-card:hover {
  transform: translateY(-4px);
  box-shadow: var(--shadow-md);
}
.room-card img {
  transition: transform 300ms ease;
}
.room-card:hover img {
  transform: scale(1.05);
}
```

### 4.3 按鈕 Loading
- 搜尋按鈕點擊後顯示 Loading 動畫
- 文字變為「搜尋中...」
- 禁用再次點擊

## 5. 圖片規格

### 5.1 Hero 背景圖
- **尺寸**：1920 × 800 px
- **格式**：WebP（優先）、JPG
- **大小**：< 300KB
- **風格**：高品質民宿外觀、自然光線、溫馨氛圍

### 5.2 房型圖片
- **尺寸**：800 × 600 px
- **格式**：WebP、JPG
- **大小**：< 100KB/張
- **風格**：明亮、整潔、寬角鏡頭

### 5.3 圖片優化
- 使用 Next.js Image 元件
- 啟用 lazy loading
- 提供適當的 sizes 屬性
